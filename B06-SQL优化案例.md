## IN改为JOIN

通过改写该句，将in改为jion

```sql
SELECT
	O.DOCNO AS DOCNO
FROM
	DOC_ATTACHMEN O
join (SELECT
			SIV2.DOCNO AS DOCNO
		FROM
			SUB_INFO_VERIFICATION SIV2
		WHERE
			SIV2.CUSTOMERID = 'ET0100000408'
		UNION
			SELECT
				O.docno AS docno
			FROM
				DOC_ATTACHMENT O
			WHERE
				O.DOCNO IN (
					SELECT
						DR.DOCNO AS DOCNO
					FROM
						DOC_RELATIVE DR,
						DOC_LIBRARY DL
					WHERE
						DR.docno = DL.DOCNO
					AND DL.DOCTYPE = '100108'
					AND DR.objectno IN (
						SELECT
							PI.progNo AS progNo
						FROM
							programmer_info PI
						WHERE
							PI.cust_id = 'ET0100000408'
					)
				)) M
ON O.DOCNO = M.DOCNO
```

## 总结

- 字符集修改为一致，性能提升8S，不是最终原因
- 通过将IN子句修改为JOIN方式问题解决
- 最外层的子句多余改写已经撤销掉
